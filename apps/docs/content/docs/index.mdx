## Introduction

The `@postiva/client` is a simple and easy to use SDK for Postiva API. It provides a set of methods to interact with the API and it's built with Typescript.

- **Easy to Use**: It is designed to be simple and easy to use.
- **Type Safety**: It is built with TypeScript to ensure type safety.
- **Developer-Friendly**: It is designed to be developer-friendly and easy to use.

## Setup

<Steps>
<Step>

### Install

```package-install
@postiva/client
```

</Step>

<Step>

### Envrionment Variables

Create a `.env` file in the root of your project and add the following:

<Callout title="Info">
For access to your API key and workspace ID, [sign up on our platform](https://dash.postiva.app/). You can locate your workspace ID via the account settings once you're registered.
</Callout>

<Tabs id="library" items={['Nextjs', 'Vite']}>
<Tab value="nextjs">
```yml title=".env"
NEXT_PUBLIC_POSTIVA_API_KEY=your-api-key
NEXT_PUBLIC_WORKSPACE_ID=your-workspace-id
```
</Tab>
<Tab value="vite">
```yml title=".env"
VITE_POSTIVA_API_KEY=your-api-key
VITE_POSTIVA_WORKSPACE_ID=your-workspace-id
```
</Tab>
</Tabs>

</Step>

<Step>

### Initialize Client

<Tabs id="library" items={['Nextjs', 'Vite']}>
<Tab value="nextjs">
```ts twoslash title="libs/postiva.ts"
declare const process: {
  env: {
    NEXT_PUBLIC_POSTIVA_API_KEY: string;
    NEXT_PUBLIC_WORKSPACE_ID: string;
  };
};
// ---cut---
import { createClient } from '@postiva/client';

export const postivaClient = createClient({
  apiKey: process.env.NEXT_PUBLIC_POSTIVA_API_KEY,
  workspaceId: process.env.NEXT_PUBLIC_WORKSPACE_ID,
});
```
</Tab>
<Tab value="vite">
```ts title="libs/postiva.ts"
import { createClient } from '@postiva/client';

export const postivaClient = createClient({
  apiKey: process.env.VITE_POSTIVA_API_KEY,
  workspaceId: process.env.VITE_POSTIVA_WORKSPACE_ID,
});
```
</Tab>
